# DemenScan - หน้าแรก (Landing Page)

## ภาพรวม

หน้าแรกของ **DemenScan** เป็นหน้า Landing Page สำหรับระบบประเมินประสิทธิภาพการทำงานของสมองด้วยตนเองสำหรับผู้สูงวัย โดยออกแบบให้ใช้งานง่าย อ่านง่าย และเป็นมิตรกับผู้สูงอายุ (Senior Friendly UI)

# UX/UI Requirements

- รองรับผู้สูงอายุ
- ตัวอักษรขนาดใหญ่ (18–20px สำหรับข้อความทั่วไป)
- Contrast สูง
- ปุ่มกดขนาดใหญ่ (อย่างน้อย 48px)
- ใช้งานง่าย ไม่ซับซ้อน
- รองรับ Touch Screen
- โหลดเร็ว
- Responsive 100%
- ใช้ภาพประกอบโทนอบอุ่น สื่อถึงผู้สูงวัยและการดูแลสุขภาพ
- ไม่มีปุ่ม "เข้าสู่ระบบ" บนหน้าแรก
- ทุกองค์ประกอบมีระยะห่าง (Whitespace) ที่เหมาะสม เพื่อให้อ่านง่ายและไม่อึดอัด
