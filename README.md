# DemenScan — เบราว์เซอร์เท่านั้น (ไม่มี backend)

ตอนนี้โปรเจกต์นี้ไม่ได้ใช้ backend อีกต่อไป: โค้ดฝั่งเซิร์ฟเวอร์ถูกลบแล้ว และแอป
จะใช้ Web Speech API ของเบราว์เซอร์สำหรับ Text-to-Speech (TTS) โดยตรง

ไฟล์สำคัญ:

- `index.html`, `style.css`, `app.js` — หน้าเว็บและตัวแอป (TTS ผ่านเบราว์เซอร์)

ถ้าต้องการคืนค่า backend เพื่อใช้ ElevenLabs อีกครั้ง ให้เพิ่มเซิร์ฟเวอร์ proxy
ที่เก็บ `ELEVENLABS_API_KEY` อย่างปลอดภัย และชี้ `TTS_ENDPOINT` ใน `app.js` ไปยัง
API ของ backend ตัวนั้น (หมายเหตุ: อย่าใส่ API key ลงในโค้ดฝั่งลูกค้าโดยตรง)

## Fallback
หากเบราว์เซอร์ไม่มี Web Speech API คุณสามารถเพิ่มสำรองด้วยไฟล์เสียงหรือ
บริการ TTS ภายนอกตามความต้องการของคุณ
